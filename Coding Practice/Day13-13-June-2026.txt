# Matrix Builds

## 🔹 Jenkins Pipeline — Matrix Build
You will learn how to **run the same build on multiple environments**.

pipeline {
    agent any
    environment {
        INSTITUTE_NAME = "Pathnex"
    }
    matrix {
        axes {
            axis {
                name 'JAVA_VERSION'
                values '8', '11', '17'
            }
        }
        stages {
            stage('Build') {
                steps {
                    sh "echo Building with Java $JAVA_VERSION"
                }
            }
        }
    }
}


#Jenkins - Fixed 

pipeline {
    agent any
    environment {
        INSTITUTE_NAME = "Pathnex"
    }
    stages {
        stage('Matrix Build') {
            matrix {
                axes {
                    axis {
                        name 'JAVA_VERSION'
                        values '8', '11', '17'
                    }
                }
            }
        }
        stage {
            stage('Build') {
                steps {
                    sh "echo Building with Java ${JAVA_VERSION}"
                }
            }
        }
    }
}

Issues fixed
1-Moved matrix inside a stage under the stages block (required in Declarative Pipeline syntax).
2-Added a wrapper stage (Matrix Build) for better pipeline organization.
3-Used ${JAVA_VERSION} in the shell command for clearer variable interpolation.
4-Improved indentation and formatting for readability.
